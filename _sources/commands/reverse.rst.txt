reverse
=======

**Name**:

geom reverse

**Description**:

geom Reverse the coordinates of the input geometry

**Arguments**:

   * -g --geometry: The input geometry

   * --help: Print help message

   * --web-help: Open help in a web browser



**Example**::

    geom reverse -g "LINESTRING (0 0, 5 5, 10 10)"
