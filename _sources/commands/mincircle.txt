mincircle
=========

**Name**:

geom mincircle

**Description**:

Calculate the minimum bounding circle of the input geometry.

**Arguments**:

   * -g --geometry: The input geometry

   * --help : Print help message



**Example**::

    geom mincircle -g "POLYGON ((0 0, 0 10, 10 10, 10 0, 0 0))"
