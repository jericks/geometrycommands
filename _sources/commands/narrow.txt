narrow
======

**Name**:

geom narrow

**Description**:

Convert a geometry collection to it's most specific type.

**Arguments**:

   * -g --geometry: The input geometry

   * --help : Print help message

   * --web-help : Open help in a web browser



**Example**::

    geom narrow -g "GEOMETRYCOLLECTION (POINT (1 1), POINT(10 10))"
