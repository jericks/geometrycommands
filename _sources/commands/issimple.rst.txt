issimple
========

**Name**:

geom issimple

**Description**:

geom Determine if the input geometry is simple or not.

**Arguments**:

   * -g --geometry: The input geometry

   * --help: Print help message

   * --web-help: Open help in a web browser



**Example**::

    geom issimple -g "POLYGON ((0 0, 0 10, 10 10, 10 0, 0 0))"
