isrectangle
===========

**Name**:

geom isrectangle

**Description**:

Determine if the input geometry is rectangular or not.

**Arguments**:

   * -g --geometry: The input geometry

   * --help : Print help message



**Example**:::

    >>>geom isrectangle -g "POLYGON ((0 0, 0 10, 10 10, 10 0, 0 0))"

    true
