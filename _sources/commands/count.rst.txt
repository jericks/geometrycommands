count
=====

**Name**:

geom count

**Description**:

geom Count the number of geometries in the input geometry.

**Arguments**:

   * -g --geometry: The input geometry

   * --help: Print help message

   * --web-help: Open help in a web browser



**Example**::

    geom count -g "MULTIPOINT (1 1, 2 2, 3 3)"
