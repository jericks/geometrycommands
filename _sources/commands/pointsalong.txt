pointsalong
===========

**Name**:

geom pointsalong

**Description**:

Place points along a line

**Arguments**:

   * -d --distance: The distance between points

   * -g --geometry: The input geometry

   * --help : Print help message

   * --web-help : Open help in a web browser



**Example**::

    geom pointsalong -g "LINESTRING (0 0, 10 10)" -d 1
