countpoints
===========

**Name**:

geom countpoints

**Description**:

Count the number of Points in the input Geometry.

**Arguments**:

   * -g --geometry: The input geometry

   * --help : Print help message

   * --web-help : Open help in a web browser



**Example**::

    geom countpoints -g "POLYGON ((1 1, 1 10, 10 10, 10 1, 1 1))"
