text
====

**Name**:

geom text

**Description**:

Create a geometry from a string.

**Arguments**:

   * -t --text: The text

   * -f --fontName: The font name

   * -s --pointSize: The font size

   * --help : Print help message

   * --web-help : Open help in a web browser



**Example**::

    geom text -t "JTS"
