endpoint
========

**Name**:

geom endpoint

**Description**:

Get the end point of a LineString or a MultiLineString.

**Arguments**:

   * -g --geometry: The input geometry

   * --help : Print help message

   * --web-help : Open help in a web browser



**Example**::

    geom endpoint -g "LINESTRING (1 1, 5 5, 10 10)"
