centroid
========

**Name**:

geom centroid

**Description**:

Calculate the centroid of a Geometry.

**Arguments**:

   * -g --geometry: The input geometry

   * --help : Print help message

   * --web-help : Open help in a web browser



**Example**::

    geom centroid -g "POLYGON ((1 1, 1 10, 10 10, 10 1, 1 1))"
