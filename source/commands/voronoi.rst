voronoi
=======

**Name**:

geom voronoi

**Description**:

Generate a voronoi diagram.

**Arguments**:

   * -g --geometry: The input geometry

   * --help : Print help message

   * --web-help : Open help in a web browser



**Example**::

    geom voronoi -g "MULTIPOINT ((12.5 12.5), (13.5 7.5), (7 9), (9.5 13), (8 7))"
