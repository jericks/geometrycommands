Name:

geom get

Description:

Get a sub geometry from a geometry collection by index.

Arguments:

  • -n –index: The index number of the Geometry
  • -g –geometry: The input geometry
  • –help : Print help message

Example::

>>>geom get -g "MULTIPOINT (1 1, 5 5, 10 10)" -n 1

POINT (5 5)
