Name:

geom distance

Description:

Calculate the distance between the two input geometries.

Arguments:

  • -o –otherGeometry: The other geometry
  • -g –geometry: The input geometry
  • –help : Print help message

Example:

geom distance -g "POINT (5 5)" -o "POINT (20 21)"
