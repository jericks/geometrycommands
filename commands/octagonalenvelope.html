Name:

geom octagonalenvelope

Description:

Calculate the octagonal envelope of the input geometry.

Arguments:

  • -g –geometry: The input geometry
  • –help : Print help message

Example:

geom octagonalenvelope -g "MULTIPOINT ((12.5 12.5), (13.5 7.5), (7 9), (9.5 13), (8 7.5))"
